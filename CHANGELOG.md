Changelog
=========

This file contains a manually maintained list of the most relevant changes for each version.
Minor bug fixes or UX changes might be missing. Use `git log` for that.

v2.2.0
-----------------

- Minimum Godot Version: 4.6
- Instead of hardcoding shortcuts, the plugin now uses the shortcuts configured for the scene tree.
- The plugin now excludes itself from the export automatically. No need to add it to the export filter anymore. (Be careful though when using json files.)

v2.1.0
------

- Adjusted the styling to fit better with the new modern theme
- Added the ability to configure the minimal width of stages
